# Swift-Fruit-Slice (AI 切水果)

基于 MediaPipe 和 OpenCV 的 AI 视觉互动切水果游戏。通过摄像头捕捉手势进行游玩，支持单手、双手以及双人 PK 模式。

## 目录结构
- `main.py`: 游戏主程序入口
- `config.py`: 游戏配置与资源加载模块
- `game_core.py`: 游戏核心逻辑与实体定义 (水果、炸弹、特效等)
- `vision_engine.py`: 基于 MediaPipe 的手部关键点追踪模块
- `math_utils.py`: 坐标平滑与防抖算法 (EWMA / Kalman)
- `ui_screens.py`: 游戏 UI 界面组件 (模式选择、刀光选择、结算界面)
- `logger.py`: 标准化日志系统
- `assets/`: 游戏图像与音效素材目录
- `models/`: MediaPipe AI 模型目录

## 环境要求与安装
确保已安装 Python 3.9+。
使用以下命令安装依赖：
```bash
pip install -r requirements.txt
```

## 运行游戏
```bash
python main.py
```

## 详细玩法说明

游戏全程支持 **AI 手势视觉交互** 与 **键盘快捷键** 双端操作。系统会自动追踪你的**食指指尖**作为交互点和切水果的刀刃。

### 1. 首页模式选择
运行游戏后，首先进入模式选择界面：
![首页选择](assets/photo/首页选择.png)

- **视觉交互**：举起一只手，将食指悬停在想要的模式方块上，进度条走满（约 3 秒）即可确认。
- **键盘快捷键**：
  - 按 `1`：选择 **单手模式**（一只手切水果）
  - 按 `2`：选择 **双手模式**（两只手同时切水果）
  - 按 `3`：选择 **双人 PK**（两人同屏对战，限时 30 秒）
  - 按 `Esc`：直接退出游戏

### 2. 选择刀光皮肤
选择完模式后，进入刀光皮肤选择界面：
![选择刀的皮肤](assets/photo/选择刀的皮肤.png)

- **视觉交互**：同样将食指悬停在喜欢的刀光特效上，等待进度条走满。
- **键盘快捷键**：
  - 按 `1`：选择 **经典冷冽刀光**
  - 按 `2`：选择 **狂暴烈焰刀光**
  - 按 `q`：快速跳过（默认使用经典冷冽刀光）

### 3. 核心切水果玩法
进入游戏后，尽情挥动你的食指来切碎屏幕下方弹出的水果吧！
- **连击系统**：连续切中水果不漏掉会增加连击数，达到一定连击数会有炫酷的全屏特效。
- **躲避炸弹**：
  - 💣 **黑色炸弹 (普通炸弹)**：切到会扣除 3 分。
    <br>![黑色炸弹](assets/zhadan/boom1.png)
  - ☠️ **红色炸弹 (致命炸弹)**：切到会直接 Game Over，结束当前对局！
    <br>![红色炸弹](assets/zhadan/boom2.png)
- **游戏结束条件**：
  - 单人模式下：漏掉的水果数量达到上限，或者切到了红色致命炸弹。
  - 双人 PK 模式下：30 秒倒计时结束，得分高者获胜。

### 4. 结算界面与重新开始
当一局游戏结束后，系统会自动计算得分（或判定双人 PK 的胜负）。
![结算选择](assets/photo/结算选择.png)

- **防误触延迟**：为了防止你在切水果时手刚好停留在按钮区域导致误触，游戏结束时会有 **2 秒的延迟**。2 秒后，交互按钮才会浮现。
- **视觉交互**：悬停食指在“重新开始”或“主菜单”方块上确认。
- **键盘快捷键**：
  - 按 `r`：快速重新开始当前模式
  - 按 `m`：返回主菜单重新选择模式
  - 按 `q` 或 `Esc`：直接退出游戏
